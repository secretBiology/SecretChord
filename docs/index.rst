SecretChord
========================================
Python library to create beautiful and robust chord diagrams.

A chord diagram is a graphical method of displaying the
inter-relationships between data in a matrix. The data are arranged
radially around a circle with the relationships between the data points typically
drawn as arcs connecting the data.
